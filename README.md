# Aboba34bot documentation
Данный бот написан для приватной группы ABOBA
Просьба прочитать данную документацию если вы являетесь её членом

Всё, что вы отправите боту, будет переслано в анонимную тему в группе

Нажав на "Начать поиск" вы перейдёте в поиск, после чего при написании текста появятся картинки прямо над строкой ввода.  
Например: "Эд Радикал" выдаст картинки Эдварда Вонг Хаул Пепелло Тевруски Четвертого.  
(Всегда будет не больше 100 картинок)  
ОДНАКО есть ограничение в около 30-50 картинок в одном ответе, по этому он разделён на страницы!  
Чтобы перейти на следующую страницу введите номер страницы через пробел в конце запроса.  
Например: "Эд Радикал 2" выдаст вторую страницу картинок (обычно больше двух страниц не будет)  

Вот особенности для удобного поиска ("спец_запрос аргумент(если нужен)" напрмиер: "file gif" или "off" или "-o" или "vpn"):
| Параметр                 | Описание                       | Возможные аргументы                                        |
| -------------            | ------------------------------ | ---------------------------------------------------------- |
| `off` `-o` `--off`       | Убирает фильтр SafeSearch (цензуру)| нет аргументов                                         |
| `vpn` `-v` `--vpn`       | Включает проксирование запроса (впн) | нет аргументов                                       |
| `site` `-s` `--site`     | На каком сайте искать          | `"pinterest.com"`, `"tenor.com"`, `"vk.ru"` и др. )        |
| `file` `-f` `--file`     | Тип возвращаемых файлов        | `"png"`, `"jpg"`, `"gif"` и др.                            |
| `image` `-i` `--image`   | Тип изображения                | `"photo"`, `"clipart"`, `"line"`, `"gif"`, `"transparent"` |
| `layout` `-l` `--layout` | Компоновка изображения         | `"square"`, `"wide"`, `"tall"` (квадратное, горизонтальное, вертикальное)                        |
| `reqion` `-r` `--region` | Географический регион поиска   |  `"wt-wt"` — всемирный (по умолч-ю), `"us-en"`, `"ru-ru"`  |
| `time` `-t` `--time`     | Время с публикамии             | `"d"` — день, `"w"` — неделя, `"m"` — месяц, `"y"` — год   |
| `color` `-c` `--color`   | Цветовое ограничение (стиль)   | `"color"`, `"monochrome"`, `"red"`, `"blue"` и др.         |

Примечание к `image`:
| Значение        | Что ищет                                    |
| --------------- | ------------------------------------------- |
| `"photo"`       | Фотографии (реалистичные изображения)       |
| `"clipart"`     | Клипарт, векторные картинки, иконки         |
| `"gif"`         | Анимированные GIF-файлы                     |
| `"transparent"` | Изображения с прозрачным фоном (обычно PNG) |
| `"line"`        | Линейные рисунки (скетчи, контуры, схемы)   |

Можно делать любые комбинации спец запросов
Например:
`@aboba34bot с днём рождения layout wide site pinterest.com color blue`
